# CS320-Software-Test

How can I ensure that my code, program, or software is functional and secure?

    * The best way I can ensure that my code, program, or softweare is functional and           secure is by writing code that uses industry best practices in the structure and          layout of the code iteslf as well as by testing early and often. Creating code that       is structure and written in a way that confirms to uniform standards is helpful for       future programers that will come into contact with my code. It will allow them to         correct or change things quickly and understand how my code is written.

How do I interpret user needs and incorporate them into a program?

      * The best way to interpret the user needs is by asking the proper questions and            having them clearly define their wants and needs. Once I am able to understand            exactly what the user is needing in a program I can then create the program to            meet those needs. 

How do I approach designing software?

      * The way I approach designing software is first by fully understanding what the            user needs in a program and how they use their existing software. By placing              myself in their shoes and listening to their needs I can create a program that            can more effectively solve their problems.
